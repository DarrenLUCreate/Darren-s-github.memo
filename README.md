### STACK & QUEUE
STACK的概念有點像堆疊書本最先放上去的是最後才被取出，最後放上去的是最先被取出，任何遞迴都可以用Stack進行改寫

First In Last Out & Last In First Out

假如說今天有個DataSet是ABC，我放入的順序是ABC，我取出來的時候是CBA
